CAPTCHA Logger v0

Data collection consent

What this extension collects

When you activate recording on a page and a CAPTCHA is detected, this extension captures:

  • CAPTCHA images (challenge imagery)
  • The page URL serving the CAPTCHA
  • Your cursor position as (x, y) coordinates over time
  • CAPTCHA position within the page (DOM selectors, bounding box)
  • Browser viewport dimensions
  • CAPTCHA prompt/instruction text (when available)
All data stays on your device. Collected data is saved as files to your local Downloads folder. This extension does not transmit any data to a server.

How recording works

Recording is not automatic. You must click the extension’s toolbar button to activate it on any given tab. Recording begins only when a CAPTCHA element is detected on the page, and stops when the CAPTCHA disappears or you navigate away.

If you accept: The extension’s recording feature will be enabled. You can activate/deactivate it per-tab via the toolbar button.

If you decline: The extension will remain installed but recording will be disabled. You can revisit this choice at any time by clicking the toolbar button.
